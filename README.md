# 运动会管理系统

## 项目概述

这是一个运动会管理系统，主要用于运动会的信息管理，包括运动员信息、比赛项目、比赛成绩等。用户可以通过主菜单-子菜单-具体操作的模式进行操作。数据存储使用结构体临时存储，并且可以利用C语言文件操作将数据导出为txt或excel文件实现备份。

## 设计概要

### 模式

运行程序后显示主菜单，用户可根据菜单输入编号，进入下一级菜单后，可以选择执行下一步操作或者返回主菜单，若执行下一步操作，所有操作结束后点击任意键自动返回主菜单，在主菜单中可以选择退出系统结束程序。

### 数据存储及备份

使用结构体临时存储运动员、学院、比赛项目、比赛成绩等信息，并且可以利用C语言文件操作将数据导出为txt或excel文件实现备份。

## 详细设计

### 菜单实现

使用 `do-while` 循环 + `switch` 语句 实现菜单的循环打印，`system("pause")` 语句使程序暂停，避免用户看不到运行结果。

### 源文件说明

- `include` 文件夹下的 `.h` 文件：各个结构体及相关操作的函数的声明，实现各个菜单的函数（view）的声明，各种头文件的引入。
- `src/model` 文件夹下的 `xxxList.cpp` 文件：对结构体变量的操作(主要是分配空间和各种get方法)的函数实现。
- `src/view` 文件夹下的 `xxxView.cpp` 文件：实现各个菜单的函数具体实现。
- `src/controller` 文件夹下的 `xxxController.cpp` 文件：对运动会管理系统的数据的增删改查操作的实现函数，将数据导出备份的函数实现(文件操作)。
- `src` 文件夹下的 `Main.cpp`：整个运动会管理系统程序的入口，调用主菜单函数进入管理系统。

## 用户手册

欢迎使用我们的运动会管理系统。本手册旨在提供系统的各项功能和操作流程的详细指导。系统包含九大模块，用户可通过输入对应的数字（1-8，0）进入各个模块进行操作。每个模块下设有主菜单，包含各种操作指令。用户可以按照系统提示，以数字形式输入指令进行操作。如果需要返回上一级菜单，只需输入0即可。

### 一、单位信息管理

1. **添加单位信息**：按照系统提示输入参赛单位名称，系统将自动为单位分配唯一的ID。
2. **删除单位信息**：输入需要删除的单位ID，系统将移除对应的参赛单位信息。
3. **修改单位信息**：输入需要修改的单位ID，系统将允许用户更新该参赛单位信息。
4. **查询单位信息**：输入需要查询的单位ID，系统将显示该参赛单位的ID和名称。
5. **返回上级菜单**：输入0，返回到主菜单。

### 二、运动员信息管理

1. **添加运动员信息**：按照系统提示输入运动员的姓名，性别，年龄和所属参赛单位ID，系统将为运动员分配唯一的ID。
2. **删除运动员信息**：输入需要删除的运动员ID，系统将移除对应的运动员信息。
3. **修改运动员信息**：输入需要修改的运动员ID，系统将允许用户更新该运动员的信息。
4. **根据运动员ID查询运动员信息**：输入运动员ID，系统将显示该运动员的详细信息。
5. **返回上级菜单**：输入0，返回到主菜单。

### 三、比赛项目管理

1. **添加比赛项目信息**：按照系统提示输入比赛项目的名称，类型，性别限定，比赛时间，场地信息，系统将为比赛项目分配唯一的ID。
2. **删除比赛项目信息**：输入需要删除的比赛项目ID，系统将移除对应的比赛项目信息。
3. **修改比赛项目信息**：输入需要修改的比赛项目ID，系统将允许用户更新该比赛项目的信息。
4. **查询所有比赛项目信息**：统将显示所有比赛项目的详细信息。
5. **根据比赛项目ID查询比赛项目信息**：输入比赛项目ID，系统将显示该比赛项目的详细信息。
6. **返回上级菜单**：输入0，返回到主菜单。

### 四、参赛信息管理

1. **添加参赛信息**：按照系统提示输入比赛ID和运动员ID，系统将记录运动员的参赛信息。若运动员已报名该项目，系统将给出提示。
2. **删除参赛信息**：输入需要删除的参赛信息ID，系统将移除对应的参赛信息。
3. **修改参赛信息**：输入需要修改的参赛信息ID，系统将允许用户更新该参赛信息。
4. **根据参赛信息ID查询参赛信息**：输入参赛信息ID，系统将显示该参赛信息的详细信息。
5. **返回上级菜单**：输入0，返回到主菜单.

### 五、秩序册自动生成

1. **秩序册预览**：系统将显示所有比赛项目以及对应的参赛运动员信息。
2. **导出秩序册到文件**：系统将将秩序册信息导出为txt文件。
3. **返回上级菜单**：输入0，返回到主菜单。

### 六、比赛成绩管理

1. **添加成绩信息**：输入运动员ID，系统将允许用户录入该运动员的比赛分数和排名。
2. **删除成绩信息**：输入运动员ID，系统将移除该运动员的比赛成绩信息。
3. **修改成绩信息**：输入运动员ID，系统将允许用户更新该运动员的比赛成绩信息。
4. **根据成绩ID查询比赛成绩**：输入成绩ID，系统将显示对应的比赛成绩信息。
5. **根据比赛项目ID查询比赛成绩**：输入比赛项目ID，系统将显示该项目下所有运动员的成绩信息。
6. **根据参赛单位ID查询比赛成绩，并汇总单位总分**：输入参赛单位ID，系统将显示该单位运动员的成绩信息，并计算单位总分。
7. **根据参赛运动员ID查询比赛成绩，并汇总运动员总分**：输入运动员ID，系统将显示该运动员的所有比赛成绩，并计算总分。
8. **返回上级菜单**：输入0，返回到主菜单。

### 七、数据导入导出

1. **数据导入功能**：系统提供数据导入功能。用户可以按照系统提示，从文件导入数据。
2. **数据导出功能**：系统提供数据导出功能。用户可以按照系统提示，将数据导出到文件。
3. **返回上级菜单**：输入0，返回到主菜单。

### 八、系统控制面板

1. **查询所有配置项**：系统将显示所有比赛规则，参赛要求，成绩选取规则等配置信息。
2. **设置运动员的最大参赛项目数量**：用户可以设置运动员的最大参赛项目数。
3. **设置比赛项目的最小参赛人数**：用户可以设置比赛项目的最小参赛人数。
4. **设置成绩名次选取规则**：用户可以设置比赛的名次选取规则，包括最少参赛人数，最大参赛人数，以及每个比赛的得分规则。
5. **返回上级菜单**：输入0，返回到主菜单。

### 零、退出管理系统

在主界面，输入0即可退出管理系统。
